# 📦 Module 2: 서비스 고도화 및 근태 관리 시스템 프로젝트
> **기간: 2026.03.26 ~ 2026.04.22**
> JPA를 활용한 데이터 무결성 확보와 사용자 편의성을 극대화한 마이페이지 및 근태 관리 시스템 구축 기록입니다.

---

### 📂 상세 기록 바로가기

#### 1. [🎓 프로젝트 회고](module2/project_review)
* **주제**: 실무형 기능을 추가한 LMS(학사관리시스템) 고도화
* **핵심 내용**: 복잡한 DB 관계 속에서의 CRUD 전략, JPA의 Update/Delete 메커니즘 이해 및 트러블슈팅 경험을 정리했습니다.

#### 2. [⚙️ 내가 맡은 기능](module2/my_role)
* **주요 역할**: 마이페이지 전반 및 관리자용 출결/연차 관리 시스템 개발
* **핵심 로직**: 
    * **Full Calendar API** 연동을 통한 실시간 근태 및 연차 신청/관리 시스템
    * **JPA 연쇄 삭제(Cascade)**를 활용한 안전한 회원 탈퇴 프로세스
    * **포인트 시스템**: 학습 활동에 따른 동적 포인트 지급 및 등급 산출 알고리즘

#### 3. [📝 학습 내용 및 주석 정리](module2/lecture_notes)
* **강의 필기**: JPA 엔티티 매핑, Spring Security 보안 설정, 동적 SQL 활용
* **주석 정리**: 데이터 무결성을 위한 제약 조건 설정과 서버 환경(IntelliJ-DB 세션) 관리 팁을 코드로 복기한 내용입니다.
